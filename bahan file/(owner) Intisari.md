## INTISARI

**ANALISIS DAN IMPLEMENTASI PADDLEOCR UNTUK EKSTRAKSI DAN VISUALISASI INFORMASI PADA DOKUMEN DRILLING REPORTS DI PT. PARAMA DATA UNIT**

Erlangga Syifa Sutrisno
22/493922/SV/20754

Pengelolaan arsip Drilling Reports di PT. Parama Data Unit saat ini menghadapi tantangan efisiensi akibat format dokumen digital yang bersifat tidak terstruktur (unstructured data). Ketergantungan pada proses rekapitulasi data secara manual tidak hanya memakan waktu, tetapi juga rentan terhadap kesalahan manusia (human error) yang dapat memengaruhi integritas data operasional. Penelitian ini bertujuan untuk mengembangkan sistem ekstraksi informasi otomatis berbasis web dengan memanfaatkan teknologi Optical Character Recognition (OCR) dan Deep Learning. Penelitian diawali dengan melakukan analisis komparasi empiris antara model EasyOCR dan PaddleOCR untuk menentukan performa terbaik dalam mengenali teks dan struktur tabel yang kompleks. Berdasarkan hasil pengujian, PaddleOCR menunjukkan akurasi yang lebih unggul dan dipilih sebagai mesin utama ekstraksi. Sistem diimplementasikan dengan mengintegrasikan modul PP-Structure untuk rekonstruksi struktur tabel, dilanjutkan dengan mekanisme pembersihan data (data cleaning), penyimpanan ke basis data, hingga visualisasi hasil pada dashboard web. Aplikasi ini diharapkan dapat mempercepat digitalisasi arsip dan mempermudah pemantauan parameter pengeboran perusahaan secara presisi.

**Kata kunci :** Drilling Reports, Ekstraksi Informasi, PaddleOCR, EasyOCR, Visualisasi Data

## ABSTRACT

**ANALYSIS AND IMPLEMENTATION OF PADDLEOCR FOR INFORMATION EXTRACTION AND VISUALIZATION ON DRILLING REPORTS DOCUMENTS AT PT. PARAMA DATA UNIT**

Erlangga Syifa Sutrisno
22/493922/SV/20754

The management of Drilling Reports archives at PT. Parama Data Unit currently faces efficiency challenges due to the unstructured nature of the digital document formats. Reliance on manual data recapitulation processes is not only time-consuming but also prone to human error, which can compromise the integrity of operational data. This study aims to develop an automated web-based information extraction system utilizing Optical Character Recognition (OCR) and Deep Learning technologies. The research begins with an empirical comparative analysis between EasyOCR and PaddleOCR models to determine the best performance in recognizing text and complex table structures. Based on the test results, PaddleOCR demonstrated superior accuracy and was selected as the core extraction engine. The system is implemented by integrating the PP-Structure module for table structure reconstruction, followed by data cleaning mechanisms, database storage, and result visualization on a web dashboard. This application is expected to accelerate archive digitization and facilitate precise monitoring of the company's drilling parameters.

**Keywords:** Drilling Reports, Information Extraction, PaddleOCR, EasyOCR, Data Visualization, PP-Structure.
